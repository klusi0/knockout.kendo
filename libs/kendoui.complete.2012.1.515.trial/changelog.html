Release notes for Kendo UI v2012.1.515

Core

What's new

  • Enhance mobile platform detection

What's fixed

  • Selection is not cleared for items which are not visible

MVVM

What's new

  • Traverse parent observable objects when resolving bindings
  • Support for setting autoBind option if bound to DataSource through MVVM
  • Multiline JSON support in data attributes

What's fixed

  • Cursor position is not persisted when value binding with update on keyup mode is used
  • The attr binding cannot set HTML5 data attributes
  • Cursor position is not persisted when value binding with update on keyup mode is used

DataSource

What's new

  • OData transport wraps the field name in tolower method to perform case insensitive filtering.

What's fixed

  • Model is not marked as dirty during its Change event
  • DataSource GetByUid does not work as expected when remote virtualization is enabled
  • Deleted item are synced even after changes are canceled
  • OData parameterMap must not append $format for json dataType
  • OData orderby parameter should not be serialized if sort is empty list
  • DataSource does not clear queued request on error
  • DataSource GetByUid does not work as expected when remote virtualization is enabled

Drag and Drop

What's fixed

  • Mousedown is prevented on drag&drop targets

Globalization

What's fixed

  • toString method rounds the number depending on the custom numeric format.
  • Improve custom numeric formatting.
  • parseFloat method does not parse strings with percent symbol correctly.

Popup

What's fixed

  • Popup close animation effects are ignored
  • Popup screen boundary detection now properly handles left and top fit overflow

FX

What's fixed

  • IE throws Access Denied when animating in a cross-domain iframe.

Templates

What's new

  • #: field# syntax for HTML encoding in Kendo templates

Validator

What's fixed

  • Multiple validation messages are shown when target input name contains dots

Web

What's new

  • Widget methods can be invoked via the jQuery plugin
  • kendo.web.js and kendo.dataviz.js can be included simultaneously

What's fixed

  • Button styling targets all input type buttons in Firefox.
  • IE throws Access Denied when animating in a cross-domain iframe.
  • Widget binding triggers the change event before setting the value.
  • Increase default drag hint z-index to overlay z-stacked windows

Upload

What's fixed

  • Cookies are not sent when during async upload on Safari
  • Selected files are uploaded after cancelling the select event
  • User data set in the upload event is duplicated on retry. Note that user data can no longer be changed on retry

Grid

What's new

  • Grid column reordering

What's fixed

  • Column resizing markers mis-positioned in hierarchy grid
  • Grid navigatable does not work on numeric/date inputs in IE9
  • Error is thrown when footer template is used and there is no data
  • Setting virtual scroll declarative does not work as expected
  • Grid sets height to scrollable container when no height is specified
  • DataItem method does not skip group footer rows
  • iPad - grouping by two columns is not performed
  • Keyboard navigation does not skip group footer rows
  • Clicking on group footer row selects it
  • Error is thrown when column with numeric textbox is focus during tabbing in IE8
  • Dragging column from a child grid can only be dropped on the parent
  • Editable grid does not pass the correct element to model's validate method.
  • items method returns footer rows
  • Grouping throws an error if page is zoomed
  • Detail template expand state is not maintained if owner group is expanded
  • Model is not passed as argument to the Save event during PopUp editing
  • Grid horizontal scrolling does not move the header cells
  • Editable grid does not pass the correct element to model's validate method.
  • items method returns footer rows

Window

What's new

  • Introduce 'appendTo' option to simplify usage in ASP.NET WebForms

What's fixed

  • Maximized windows with title are higher than the page
  • Setting small font-size breaks titlebar buttons rendering
  • Bottom border is invisible in IE7
  • Content iframe gets removed if window is refreshed with local URL
  • Modal Window remains below overlay after closing another modal Window
  • toFront() increases the window z-index even if the window is topmost
  • Modal Window remains below overlay after closing another modal Window

DatePicker

What's new

  • Support for case sensitive searching

What's fixed

  • Binding the widget to ViewModel sets its options to default ones.
  • Calendar navigates to incorrect view if current date is 1.1.2012

Slider

What's fixed

  • Values() doesn't work with array of numbers.
  • Slider tooltip format does not work.
  • Tooltip of the slider is not visible when the slider is in the window.
  • Calling enable() on an enabled slider causes unexpected behavior.

AutoComplete

What's new

  • Widget lowers the filter value when perform case insensitive filtering.
  • Support for case sensitive filtering

What's fixed

  • Unable to select item in BM:IE9 DM:IE7 mode.
  • AutoComplete's value method returns placeholder value in IE
  • Widget moves the cursor to the begging on input focus.
  • Popup width is incorrectly calculated for initially hidden widgets
  • Change event was not raised on blur

ComboBox

What's new

  • Widget lowers the filter value when perform case insensitive filtering.
  • ComboBox copies the CSS classes to the wrapper element.
  • Support for case sensitive filtering
  • Support for case sensitive selection using the text method

What's fixed

  • Unable to select item in BM:IE9 DM:IE7 mode.
  • Widget moves the cursor to the begging on input focus.
  • Popup width is incorrectly calculated for initially hidden widgets
  • Multiple ComboBoxes with the same DataSource and autoBind=false render popup simultaiously when one of them is activated

DropDownList

What's fixed

  • Empty DropDownList falls into an infinite loop when try to open it.
  • Unable to select item in BM:IE9 DM:IE7 mode.
  • DropDownList unbinds users' event handlers when disable the widget.
  • Popup width is incorrectly calculated for initially hidden widgets
  • Keyboard navigation of the DropDownList does not work after rebind.

Editor

What's new

  • Document all available editor methods

What's fixed

  • Indenting plain text generates <div> element
  • Disallowing clipboard access inserts BOM characters when pasting
  • The 'encoded' property is not honored when triggering change event
  • Empty paragraphs are unfocusable in Chrome/FF when initializing from textarea content
  • Pasting content twice throws JS error in IE
  • Support mobile devices with contentEditable besides iOS 5.x
  • Cannot unbullet multiple lists within a table
  • Changing list type of lists in tables causes loss of data

TreeView

What's new

  • Observable objects cannot be used as node data when dynamically inserting nodes

What's fixed

  • DragStart event cannot be prevented
  • Dynamically inserted nodes do not use the treeview template

Menu

What's new

  • Improve item creation from plain HTML
  • Add popupCollision property

What's fixed

  • Menu links are not navigating on mobile.
  • Menu doesn't support content in dataSource

NumericTextBox

What's fixed

  • Uppercase format letters are not supported
  • Formatted values are not shown in iPad (iOS 5)

TabStrip

What's fixed

  • TabStrip doesn't work on mobile device, when the mobile scroller script is not loaded

ListView

What's new

  • Expose Pager change event

What's fixed

  • Selectable and navigatable are not present in ListView options
  • Constrain loading icon to the widget container

PanelBar

What's new

  • Add activate event, fired after expand complete

What's fixed

  • PanelBar doesn't handle well the expanded data property.
  • PanelBar doesn't show content element and arrows with dataSource init.
  • PanelBar not showing arrows for added items with content.
  • Fire collapse event when expanding in single mode

Splitter

What's fixed

  • Focus of inputs inside panes is sometimes prevented

DataViz

What's new

  • Widget methods can be invoked via the jQuery plugin
  • kendo.web.js and kendo.dataviz.js can be included simultaneously

What's fixed

  • Gauge svg method throws an error

What's fixed

  • seriesHover is not triggered when moving over neighboring points in line/area series
  • Clicking on a data point label text does not trigger a seriesClick event
  • Gradient fills are rendered as black if base tag is present on the page
  • Transparent background is rendered as black in Opera
  • Error when binding to a field that is not defined in the data item
  • Points with null X or Y value should be treated as null
  • Background is set to black when specifying transparent (VML)
  • dashType is not applied to scatter line series
