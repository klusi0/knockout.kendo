Area Charts

Area Chart emphasizes changes in values by filling in the portion of the graph beneath the line that connects each data point in a series.

Bar Charts

Show value changes over time, or illustrate a quantitative comparison of multiple entities.

Pie Charts

Show the relative quantitative value or percentage of items with respect to the total.

Line Charts

Show trends in data over regular intervals, or show trends among multiple entities.

Scatter Charts

Show the relationship of multiple values or entities, often grouped by category or relationship, along a common x- and y-axis.

Multiple Axes

Multi-axis charts enable comparison of a set of entities based on multiple parameters.

Radial Gauge

Show range or progress data in a radial (circular or semi-circular) view. Useful for illustrating progress toward a goal, the current status of some value within a range of upper and lower bounds, or a summary of some fluctuating metric.

Linear Gauge

Show range or progress data in a linear, thermometer-like view. Useful for illustrating progress toward a goal, the current status of some value within a range of upper and lower bounds, or a summary of some fluctuating metric.
